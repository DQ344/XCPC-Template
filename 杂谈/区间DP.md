# 区间DP

常规的区间DP有以下两种：
1. 枚举断点，将当前区间划为两区间，进行合并
2. 将新数放入队头/尾进行考虑

我们将 $f(l, r)$ 定为将区间 $(l, r)$ 操作成我们需要的区间的花费

对于 $1$ 来说，$f(l, r) = min(f(l, k) + f(k + 1, r) + cost)$
对于 $2$ 来说，$f(l, r) = min(f(l + 1, r - 1), f(l, r - 1), f(l + 1, r)) + cost$

$1$ 复杂度为 $O(n^3 · opt(cost))$

$2$ 复杂度为 $O(n^2 · opt(cost))$
